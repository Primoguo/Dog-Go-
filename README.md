# 自律狗

自律狗是一款面向当代自律人群的轻量自律陪伴产品。首发 MVP 聚焦 iOS，以健身为默认场景，同时覆盖学习和作息。核心假设是：用户不是缺少任务列表，而是缺少一个有性格、有反馈、有关系感的陪伴角色，帮助他们在中断后温和回到节奏。

## Project Structure

- `docs/`：产品文档、PRD、决策记录。
- `backlog/`：用户故事、epic 拆解、开发任务。
- `research/`：用户访谈、竞品观察、验证记录。
- `design/`：设计 brief、用户流程、线框图、视觉方向。
- `app/`：后续放 iOS 或移动端 Web 实现。
