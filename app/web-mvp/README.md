# 自律狗 Web MVP

这是一个静态移动端 Web 原型，用于验证自律狗核心创意：

领养狗狗 → 创建目标 → 首页狗狗小世界 → 完成打卡 → 狗狗反馈 → 查看进度。

## 打开方式

直接在浏览器打开：

```text
app/web-mvp/index.html
```

## 已实现

- 4 只首发狗狗选择：柴犬、金毛、边牧、中华田园犬。
- 目标创建：健身、学习、作息模板。
- 首页狗狗小世界。
- 根据目标类型切换道具标签。
- 完成今日打卡。
- 狗狗反馈页。
- 亲密度、等级、饱腹、清洁、精力更新。
- 进度页。
- 本地 `localStorage` 保存状态。
- 模拟漏一天，验证恢复任务体验。
- 重置体验。

## 当前限制

- 首版只有一张小院子插画底图。
- 狗狗品种目前通过卡片和文案区分，首页底图里的狗狗还没有按品种切换。
- 恢复状态通过“模拟漏一天”触发。
- 未接后端、登录、推送或真实日期任务调度。
